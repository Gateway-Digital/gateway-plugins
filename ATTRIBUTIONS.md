# Attributions

gateway-core adapts patterns and selected content from these MIT/Apache-licensed
open-source projects. When cherry-picking further content (per the find-org-skills
skill), add a line here with the source commit.

- **Superpowers** (obra/superpowers, MIT) — the SessionStart method-injection hook
  pattern, skill-enforcement language, brainstorming/plan-writing discipline
  (no-placeholders, task right-sizing), subagent-driven execution (file-based briefs,
  dual review gate, progress ledger, base-commit diff rule), systematic-debugging
  phases and iron law, verification-before-completion.
- **Everything Claude Code** (affaan-m/ECC, MIT) — agent definitions adapted for
  planner, code-explorer, code-reviewer (confidence gating, false-positive filters,
  zero-findings-valid), security-reviewer, build-fixer; save/resume-session command
  pattern; model right-sizing table; rules taxonomy; mgrep recommendation.
- **spec-kit** (github/spec-kit, MIT) — spec structure (FR-/SC- IDs, Given/When/Then,
  priority-tiered user stories), [NEEDS CLARIFICATION] with 3-marker cap, clarify's
  recommended-option question tables, constitution pattern, complexity-justification
  gate, spec quality checklist.
- **BMAD-METHOD** (bmad-code-org/BMAD-METHOD, MIT) — scale-adaptive routing rubric,
  one-session-per-workflow discipline, project-context capture for brownfield.
- **AI Engineering Coach** (microsoft/AI-Engineering-Coach, MIT) — the AI-readiness
  checklist used in org-init and the anti-pattern vocabulary in training docs.
- **vercel-labs/skills** (MIT) — the find-skills discovery pattern and quality gates,
  adapted into find-org-skills.
